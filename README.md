# Simple Todo App

A clean, lightweight task management app that helps you stay on top of what needs to get done — and by when.

## Functionality

The Simple Todo App lets you manage your daily tasks with ease:

- **Add tasks** — Click the "+ Add Todo" button to open a form and enter a new task
- **Set due dates** — Optionally specify a deadline for each task
- **Complete tasks** — Check off todos as you finish them
- **Track progress** — A live counter at the top shows how many tasks you've completed out of the total

The interface is intentionally minimal: no accounts, no syncing, just a straightforward list to keep you focused.

## Technology

This project is built with core front-end technologies:

- **HTML5** — Semantic markup and structure
- **CSS3** — Styling and layout, including responsive design
- **JavaScript (ES6+)** — DOM manipulation, event handling, and state management
- **GitHub Pages** — Static site hosting and deployment

No frameworks or external libraries were used — this is a vanilla JS project built from scratch as a practice exercise in working with the DOM and managing UI state.
